Software para identificação de anúncios de uma determinada página para o Jornal. 
Objetivo do Software é captar as imagens dada para o software, analisar se é existente no PDF, caso exista conceder o : tamanho, localização na página, conversões(pixel, dpi, etc).

Para utilizar o Software é necessário ter o python instalado no PC, executar o .exe e também baixar as depêndencias encontrada no .txt de "requeriments"







pip install PyMuPDF Pillow opencv-python-headless imagehash scikit-image numpy
